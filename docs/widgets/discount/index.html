Discount 插件说明
  • id

    这是有 Bothub 自动生成的编号，你需要将带有此编号的元素放入您的网页中，SDK 将会自动将其按照配置渲染成为对应的插件。

  • type

    这是一个固定的值Discount，表示此插件的类型。

  • pageId

    Facebook 页面 ID，表示您的 Bot 所连接的 Facebook 页面的编号。

  • origin

    插件加载网址的基域，必填。

    指示插件在哪个网站加载，如果不填该项，插件将会无法显示。

    在这里我们将其写死成当前网页，作为展示用。

  • title

    优惠券标题

  • subtitle

    优惠券副标题

  • discountText

    折扣优惠码的提示文本

  • discountCode

    折扣优惠码

  • showCodeBtnText

    折扣按钮文本

  • copyCodeBtnText

    复制按钮文本

  • discount

    折扣

  • hideAfterChecked

    用户勾选确认后多少天内自动隐藏，选填。

    默认不启用此功能。

  • align

    插件对齐方式，选填。

    默认为居中。

    • 预览 刷新插件
    插件加载网址的基域 标题 副标题 折扣优惠码的提示文本 折扣的优惠码 折扣按钮文本 复制按钮文本 折扣数量 选中后隐藏 (选填) 对齐方式 (选填)
