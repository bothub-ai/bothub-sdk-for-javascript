插件说明和预览
  • id

    这是有 Bothub 自动生成的编号。

  • type

    这是一个固定的值Customerchat，表示此插件的类型。

  • pageId

    Facebook 页面 ID，表示您的 Bot 所连接的 Facebook 页面的编号。

  • themeColor

    主题颜色，表示插件的主要背景色，选填。

    应填入合法的颜色编码，如#FF9966或rgba(100, 100, 200, .5)

  • loggedInGreeting

    对当前已登录 Facebook 的用户显示的欢迎语，选填。

    不超过 80 个字符。

  • loggedOutGreeting

    对当前未登录 Facebook 的用户显示的欢迎语，选填。

    不超过 80 个字符。

    • Customerchat 预览 刷新插件
    主题颜色 (选填) 登录用户欢迎语 (选填) 未登录用户欢迎语 (选填)
